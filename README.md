# Trote Solidário 2026

Projeto acadêmico com uma landing page simples para divulgação do evento **Trote Solidário 2026** da UniFACEF.

## Estrutura

- `index.html`: página principal.
- `assets/css/main.css`: estilos da aplicação.

## Como executar

Abra o arquivo `index.html` em qualquer navegador moderno.

## Melhorias aplicadas

- Padronização de nomes de arquivos e pastas para formato profissional.
- Correção do caminho do CSS no HTML.
- Ajustes semânticos e de acessibilidade básicos no HTML.
- Limpeza e organização dos estilos CSS.
