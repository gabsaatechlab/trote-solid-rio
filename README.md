# trote-solid-rio

Projeto reorganizado como uma landing page acadêmica simples para apresentar a proposta do Trote Solidário 2026.

## Estrutura atual

- `index.html`: página principal do projeto
- `style.css`: identidade visual e layout da landing page

## Objetivo

Melhorar a apresentação de um projeto inicial, deixando o repositório mais limpo, com nomes padronizados e uma leitura mais profissional.
